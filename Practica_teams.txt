EJERCICIO 4: Complejidad de método
Consigna: Determiná la familia de complejidad temporal del siguiente método en el peor caso. Justificá tu respuesta aplicando las reglas de suma y simplificación vistas en la cátedra.
public static void procesoMezclado(int[] datos) {
    int n = datos.length;

    // Bloque A
    for (int i = 0; i < n; i++) {   // N_1
        System.out.println(datos[i]); // C1: Constante
    }

    // Bloque B
    for (int i = 0; i < n; i++) {   //N_2 peor caso!
        for (int j = 0; j < n; j++) {   //N_3
            datos[i] = datos[i] + datos[j]; // C2: Constante
        }
    }
}
